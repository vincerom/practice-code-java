Sort Algorithm

1. Bubble Sort
  Time:
    Best: O(n^2)
    Worst: O(n^2)
    Average: O(n^2)
  Space: O(1)
  Stable: true

2. Insert Sort
  Time:
    Best: O(n^2)
    Worst: O(n^2)
    Average: O(n^2)
  Space: O(1)
  Stable: true

3. Quick Sort
  Time:
    Best: O(nlogn)
    Worst: O(n^2)
    Average: O(nlogn)
  Space: O(logn)
  Stable: false

4. Selection Sort
  Time: O(n^2)
  Space: O(1)
  Stable: false

5. Merge Sort
  Time:
    Best: O(nlogn)
    Worst: O(nlogn)
    Average: O(nlogn)
  Space: O(n)
  Stable: true
